Design System — SubtitleStudio
Fonte única de verdade para cor, tipografia, espaçamento, componentes e regras de UX.
Use este documento como contexto ao pedir qualquer ajuste de interface ao Cursor ou outro agente de código.
1. Filosofia e tom visual
Dark-first, sem gradientes
O app é 100% dark mode. Nenhuma tela tem fundo claro. Proibido: gradientes, sombras decorativas, glow, blur, glassmorphism, texturas. Superfícies são sólidas e planas.
Verde como única cor de ação
O verde #1D9E75 é a cor de marca e de ação primária. Botões de confirmação, progress bar, badges de sucesso, foco de input e seleção de linha usam esta cor. Nenhuma outra cor deve competir com ela em botões de ação.
Hierarquia por opacidade, não por cor
Texto primário #e8e8e8, secundário #909090, mudo #606060. Nunca use branco puro #fff para texto em superfície — reservado para texto sobre fundo colorido (ex: botão primário).
Bordas finas, não containers pesados
Bordas são sempre 0.5px. Nenhum elemento tem sombra de caixa para separação — use diferença de nível de superfície (bg-page → bg-surface → bg-elevated).
2. Paleta de cores
Todas as variáveis CSS devem ser declaradas em :root no arquivo global de estilos (globals.css ou tokens.css). Nunca use hex fixo no código dos componentes — sempre referencie a variável.

15. Regras de implementação para o Cursor
Sempre use variáveis CSS, nunca hex fixo nos componentes
Todos os valores de cor devem referenciar as variáveis de :root. Hex direto só em globals.css / tokens.css onde as variáveis são declaradas.
Tamanho de ícone sempre explícito
Todo SVG de ícone deve ter width e height definidos diretamente: botões inline = 13–14px, sidebar = 16px, decorativo = 20px máximo. Nunca herde font-size.
Bordas sempre 0.5px
Nenhuma borda usa 1px ou mais, exceto o focus ring. border: 0.5px solid var(--border) é o padrão universal.
Fonte sempre explícita em elementos de formulário
Inputs e selects não herdam a fonte do body em todos os browsers. Sempre adicione font-family: inherit ou font-family: var(--font).
Timestamps no formato YYYY-MM-DD HH:mm:ss
Todos os timestamps exibidos no app usam este formato UTC sem fuso. Usar date.toISOString().replace('T',' ').slice(0,19) ou equivalente.
Checkbox e radio: accent-color: var(--accent)
Todo input de seleção usa a cor de acento verde. accent-color: var(--accent) sempre.
Proibido: gradientes, sombras decorativas, glassmorphism
background: linear-gradient(...) · box-shadow decorativo · backdrop-filter: blur() · opacity em texto (use a cor certa) · cores fora da paleta definida acima.
Proibido: font-weight acima de 600
O app usa weight 400 (regular), 500 (destaque) e 600 (título de página apenas). Nunca use 700, 800 ou 900.
